# Negociar con Japón · 交渉
## Presentación Interactiva

Guía de protocolo, cultura y negociación para empresas españolas que operan en el mercado japonés.

### 🚀 Deploy en GitHub Pages

1. Crea un nuevo repositorio en GitHub (ej: `japon-presentacion`)
2. Sube todos los archivos de esta carpeta:
   - `index.html`
   - `video.mp4`
   - `README.md`
3. Ve a **Settings → Pages → Branch: main → Save**
4. Tu presentación estará en: `https://TU-USUARIO.github.io/japon-presentacion`

### 📁 Archivos necesarios
- `index.html` — La presentación completa
- `video.mp4` — Video de fondo (5.7MB, compatible con GitHub)
